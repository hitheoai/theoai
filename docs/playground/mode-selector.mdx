---
title: "Mode Selector"
description: "Choose execution modes in the Playground."
---

The mode selector lets you control which Theo engine handles your request:

| Mode | What Happens |
|------|-------------|
| Auto | Theo classifies your prompt and picks the best engine |
| Fast | Routes to Theo Flash — quick, cheap |
| Think | Routes to Theo Reason — deep reasoning |
| Code | Routes to Theo Code — optimized for code |
| Research | Runs async web research with citations |
| Image | Routes to Theo Create — image generation |
| Roast | Routes to Theo Edge — unfiltered humor |
