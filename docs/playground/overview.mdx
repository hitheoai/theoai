The Playground at [hitheo.ai/playground](https://hitheo.ai/playground) is an interactive environment for testing Theo's capabilities.

## Features

- **Mode selector** — switch between auto, fast, think, code, research, image, roast
- **Skill activation** — enable/disable installed skills
- **Real-time streaming** — see tokens as they're generated
- **Tool execution** — watch tools being called in real-time
- **Artifact viewer** — preview generated images, code, and documents
- **Cost tracking** — see per-request cost in real-time
- **Model info** — see which model was selected and why
