The Skills page is your central hub for managing AI capabilities.

## Marketplace

Browse the skill marketplace to discover domain-specific AI capabilities:

- **Search** by name, category, or keyword
- **Filter** by category: Productivity, Domain, Integration, Automation, Creative
- **Sort** by install count, rating, or newest
- **Preview** skill details, permissions, and reviews before installing

## Installed Skills

Manage your installed skills:

- **Toggle on/off** — Disable a skill without uninstalling it
- **View details** — See the skill's system prompt, tools, and permissions
- **Uninstall** — Remove the skill entirely
- **Configure** — Adjust skill-specific settings (where available)

Installed skills are automatically loaded on every completion request. You can also activate skills per-request using the `skills` parameter.

## For Skill Authors

- **My Skills** — View skills you've authored
- **Publish** — Submit skills for marketplace review via CLI (`theo skill publish`) or API
- **Analytics** — Track install counts and usage
- **Earnings** — Monitor revenue from the [revenue sharing](/skills/revenue-sharing) program

See [Building a Skill](/skills/building-a-skill) to get started as an author.
